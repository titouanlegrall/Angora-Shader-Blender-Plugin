# 🎨 Angora Shader - v0.0.1 (Pre-Alpha)



A high-end Non-Photorealistic Rendering (NPR) toolkit for **Blender 5.0.0**. Instantly transform 3D meshes into living paintings with dynamic geometry-based brush strokes.

## 🌟 Overview

**Angora Shader** bridges the gap between 3D precision and 2D aesthetics. By leveraging **Geometry Nodes** and a custom **Toon Shader**, it scatters physical "paint strokes" across your model that react to light, camera angles, and object normals.

## ✨ Key Features

- 🖌️ **Dynamic Stroke Scattering:** High-performance instancing of brush textures.
- 🌈 **Advanced Toon Shader:** Full control over Base, Shadow, and Light Blend colors.
- 🖼️ **Visual Brush Selector:** Real-time preview grid for swapping textures.
- 🎥 **Camera-Aware Orientation:** Strokes dynamically face the camera for a consistent 2D look.
- ⚡ **One-Click Workflow:** Dedicated N-Panel for instant application.

## 🛠️ Installation

1. Download the `AngoraShader_v0.0.1.zip` from the [Releases](https://github.com/your-profile/Angora-Shader/releases) page.
2. In Blender, go to **Edit > Preferences > Add-ons**.
3. Click **Install...** and select the `.zip` file.
4. Enable **Object: Angora Shader**.

---

## 📖 Quick Start Tutorial

1. **Select** your target mesh.
2. Open the **Sidebar (N-Key)** and locate the **Angora** tab.
3. Click **"Apply Angora Shader"**.
4. **Customize:**
   - Change the brush style using the **Texture Preview** grid.
   - Adjust **Density** and **Scale** in the Modifier Properties.
   - Tweak lighting response in the Shader Editor.
